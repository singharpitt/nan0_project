# Sorting Visualizer

**This Nano Project consists of a website that visualizes the sorting algorithms**

***List of Sorting Algorithms in this repository***
```
Bubble Sort
Selection Sort
Insertion Sort
Quick Sort
Merge Sort
```

# Get Started 
## Installation 

```
npm install
npm start
```
